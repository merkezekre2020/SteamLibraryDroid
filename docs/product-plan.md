# Product Plan

## Ürün amacı

Steam kütüphanesini mobilde düzenli, filtrelenebilir ve kişisel notlarla yönetilebilir hale getirmek.

## Hedef kullanıcı

- Steam oyuncuları
- Backlog yöneten kullanıcılar
- Oynadığı oyunlara puan/not vermek isteyenler

## MVP özellikleri

- E-posta ile auth
- Steam profil bağlama
- Profil URL veya Steam ID ile import
- Oyun listesi
- Arama, temel filtreleme
- Oyun detay ekranı
- Durum: backlog, playing, completed, dropped
- Favori, puan, not
- Son senkron bilgisi

## V2 fikirleri

- Achievement görüntüleme
- İstek listesi
- Fiyat takibi
- İstatistik ekranı
- Koleksiyon paylaşımı
